# Frontend Screenshots

This directory contains screenshots generated by Playwright E2E tests.

## How to Generate Screenshots

Run the following command to generate all screenshots:

```bash
npx playwright test
```

This will create screenshots for:
- Homepage (desktop and mobile)
- Products page
- Product detail page
- Cart page
- Login page
- Register page
- Form validation states
- Mobile menu

## Screenshot Files

After running tests, you'll find:
- `homepage.png` - Landing page with hero section
- `products-page.png` - Product listing grid
- `product-detail.png` - Individual product view
- `empty-cart.png` - Empty cart state
- `login-page.png` - Login form
- `register-page.png` - Registration form
- `mobile-header.png` - Responsive mobile header
- `mobile-menu-open.png` - Mobile navigation menu
- And more...

## Viewing Results

Screenshots are automatically captured during test execution. To view the test report with screenshots:

```bash
npx playwright show-report
```
